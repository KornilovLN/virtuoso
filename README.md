## Тестовый проект работы с Virtuoso

#### За базу взят официальный образ и созданы:

* Dockerfile
* virtuoso.ini
* rebuild.sh
* start_virtuoso.sh

#### Запуск на локальном компьютере:

* Стартовать файл rebuild.sh, если рабочего образа еще нет.
* start_virtuoso.sh
* Войти в браузере на localhost:8890
